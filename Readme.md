# Git Proyecto Practica

## Aprendizaje

Documentación: Espacio de Práctica para Git

Descripción:
Este directorio está designado como un área segura para practicar y desarrollar 
habilidades de control de versiones con Git sin riesgo de afectar código en producción.

Propósito:

- Experimentar con comandos de Git
- Crear ramas y realizar merge
- Practicar resolución de conflictos
- Aprender flujos de trabajo colaborativos
